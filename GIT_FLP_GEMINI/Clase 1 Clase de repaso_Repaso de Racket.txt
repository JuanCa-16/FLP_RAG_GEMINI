Es un lenguaje de programación multiparadigma, que trabaja diferentes paradigmas de programación Funcional, Procedural, Objetos, Eventos. Vamos a trabajar en el enfoque funcional.

###
Racket utiliza notación prefija, donde primero va la función y después los argumentos, usando una notación basada en paréntesis. Aquí se muestra cómo declarar el lenguaje y definir variables:

```racket
#lang eopl
(define pollito 5)
(define gaillinita (* pollito 2))
```

###
Las funciones en Racket se declaran con `lambda`, ya que las funciones serán tratadas como valores. También se pueden definir parámetros con valores por defecto.

```racket
(define funcion
  (lambda (x y)
    (+ x y)))

(funcion 1 2)

;Funciones con parametro por defectos
(define funcionPro
  (lambda (x [p 8])
    (+ x p)))

(funcionPro 4) ;12
(funcionPro 4 9) ;13
```

###
Para la toma de decisiones, se utilizan estructuras condicionales como `cond`:

```racket
(define funcion
    (lambda (x y)
        (cond
            [(> x 5) (+ x y)]
            [(and (> x 0) (< x 5)) (* x y)]
            [else (+ x y 2)])))
```

###
Las listas son una estructura de datos fundamental en Racket. Se pueden crear de varias maneras, incluyendo `list`, `cons` para construir desde cero, o con la notación de comilla simple para listas literales:

```racket
(list 1 2 3 5)
(cons 1 (cons 2 (cons 3 (cons 5 empty))))
'(1 2 3 4 a b (1 2 3))
(list 1 2 3 'a 'b (list 1 2 3))
```

###
Para acceder a los elementos de una lista, se utilizan funciones como `car` (para la cabeza o primer elemento) y `cdr` (para la cola o el resto de la lista). A partir de estas, se derivan otras como `cadr` y `caddr` para acceder a elementos específicos.

```racket
(define l '(1 2 3 4 5))
(car l) ; 1
(cdr l) ; '(2 3 4 5)
(car (cdr l)) ; 2
(car (cdr (cdr l))) ; 3
(cadr l) ; car del cdr
(caddr l) ; car del cdr del cdr
```

###
Las listas se procesan comúnmente con funciones recursivas. Aquí hay ejemplos de cómo sumar los elementos de una lista, primero de forma directa y luego usando un acumulador para optimización:

```racket
(define sumarL
  (lambda (lst)
    (cond
      [(null? lst) 0]
      [else (+ (car lst) (sumarL (cdr lst)))])))

(define sumarColita
  (lambda (lst [acc 0])
    (cond
      [(null? lst) acc]
      [else (sumarColita (cdr lst) (+ acc (car lst)))])))
```
