La secuencia de cálculos es la siguiente:
1.  Se define una expresión inicial que resulta en `2y = 5`.
2.  Posteriormente, se establece `x = 6`, lo que lleva a que `y` también tome el valor de `6`.
3.  La suma de `6 + 5` da como resultado `11`.
4.  El valor de `x` se confirma como `6`.
5.  La suma `11 + 6` produce `17`.
6.  Se asigna `y = 8`.
7.  La operación `7 + 8` resulta en `25`.
8.  Finalmente, se establece `x = 4`. Se realiza la operación `7 - 4 = 3`. Este resultado se multiplica por `25`, dando `3 * 25 = 75`.

*   Se definen variables iniciales: `let x`, `let x6`, `x7`.
*   Un `port` de definiciones configura el área de trabajo, comenzando con `let x8`.
*   El área de trabajo incluye una expresión `más let`, donde `x6` se iguala a `x`, y se realiza la operación `x + x`.
*   Se define `let y3`, aplicando `x` e `y` sobre una expresión.
*   Posteriormente, se introduce otra expresión `más x`, la cual puede simplificarse a `x dot y`.

*   Se incluye una definición para `y` con su valor.
