####
Definición de constantes y predicados para la representación de números en base 16:

```racket
#lang eopl
(define base 16)
(define zero '())
(define zero? null?)
```

**Conceptos clave**
`eopl`, `constantes`, `predicados`, `listas`, `representación de datos`

**Aplicación**
Establece la infraestructura base para implementar aritmética posicional. El uso de `null?` como predicado para el cero permite identificar listas vacías como el valor numérico base dentro del sistema definido.

####
Implementación de la función `succ` para calcular el sucesor de un número:

```racket
(define succ
  (lambda (n)
    (cond
      [(zero? n) (list 1)]
      [(= (car n) (- base 1)) (cons 0 (succ (cdr n)))]
      [else (cons (+ 1 (car n)) (cdr n))])))
```

**Conceptos clave**
`recursión`, `condicionales`, `listas`, `aritmética`

**Aplicación**
Esta función incrementa en una unidad el valor representado por la lista. Gestiona el acarreo cuando el dígito actual alcanza el valor de `base - 1`, permitiendo la propagación del incremento a través de la representación numérica.

####
Implementación de la función `pred` para calcular el predecesor de un número:

```racket
(define pred
  (lambda (n)
    (cond
      [(zero? n) (eopl:error "El número no puede ser cero")]
      [(= (car n) 0) (cons (- base 1) (pred (cdr n)))]
      [(equal? (succ zero) n) '()]
      [else (cons (- (car n) 1) (cdr n))])))
```

**Conceptos clave**
`manejo de errores`, `recursión`, `predecesor`, `condicionales`

**Aplicación**
Decrementa un valor numérico definido como lista. Incluye lógica de control para el caso base, reportando un error mediante `eopl:error` si se intenta obtener el predecesor de cero o ajustando los dígitos mediante un préstamo cuando el valor actual es cero.

####
Implementación de la suma aritmética mediante recursión:

```racket
(define plus
  (lambda (x y)
    (if
     (zero? y)
     x
     (succ (plus x (pred y))))))
```

**Conceptos clave**
`recursión`, `abstracción funcional`, `funciones de orden superior`, `aritmética`

**Aplicación**
Define la operación de suma entre dos números definidos previamente. Utiliza las funciones `succ` y `pred` para transformar la operación de adición en una serie de incrementos y decrementos recursivos hasta que el segundo operando llega al estado `zero`.
