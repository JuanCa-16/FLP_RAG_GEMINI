######## RESUMEN ########
Implementa aritmética básica sobre números grandes representados como listas de dígitos en base hexadecimal.

######## CONCEPTOS ########
Abstracción de datos, Recursión, Aritmética de números grandes, Representación posicional, Operaciones sobre listas

######## EXPLICACION ########
Este código define un sistema para manipular números enteros arbitrariamente grandes (BigNum) usando una representación personalizada.
Permite a los programadores realizar operaciones fundamentales como la suma (`plus`) y la multiplicación (`mult`) con números que exceden los límites de un tipo de dato entero estándar.
Demuestra cómo construir funciones de incremento (`successor`) y decremento (`predecessor`) que manejan el acarreo y el préstamo entre dígitos.
Aplica una fuerte abstracción de datos al representar números como listas y utiliza recursión mutua para implementar las operaciones aritméticas, basándose en el principio de dígitos de menor peso primero.
Utiliza la base 16 para ilustrar el funcionamiento de sistemas numéricos posicionales, fundamentando la comprensión de cómo se implementan tipos de datos numéricos complejos.

######## CODIGO ########
#lang eopl
(define base 16)

(define zero '())
(define is-zero? null?)

(define successor
  (lambda (n)
    (cond
      [(is-zero? n) (cons 1 empty)]
      [(= (car n) (- base 1))
       (cons 0 (successor (cdr n)))]
      [else
       (cons (+ (car n) 1) (cdr n))])))

(define predecessor
  (lambda (n)
    (cond
      [(is-zero? n) (eopl:error "No puede manejar predecesor de cero")]
      [(equal?  n (cons 1 empty)) '()]
      [(= (car n) 0)
       (cons (- base 1) (predecessor (cdr n)))]
      [else (cons (- (car n) 1) (cdr n))]
       )))

;;Area del programador
(define plus
  (lambda (a b)
    (cond
      [(is-zero? b) a]
      [else
       (plus (successor a) (predecessor b))]
      )
    ))

(define mult
  (lambda (a b)
    (if
     (is-zero? b)
     zero
     (plus a (mult a (predecessor b)))
     )))
