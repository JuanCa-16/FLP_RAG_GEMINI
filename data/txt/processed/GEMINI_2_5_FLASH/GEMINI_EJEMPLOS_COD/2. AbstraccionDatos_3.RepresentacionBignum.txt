######## RESUMEN ########
Implementa aritmética de números grandes (bignums) mediante una representación de datos en listas.

######## CONCEPTOS ########
Bignums, Abstracción de Datos, Recursión, Aritmética Posicional

######## EXPLICACION ########
Este código implementa una representación y operaciones aritméticas básicas para números enteros arbitrariamente grandes (bignums) en Racket.
Permite manipular y realizar cálculos con números que sobrepasan los límites de los tipos de datos numéricos nativos del lenguaje, como en aplicaciones de criptografía o teoría de números.
Ilustra cómo construir tipos de datos complejos a partir de estructuras más simples, facilitando el diseño de bibliotecas matemáticas personalizadas.
Emplea una representación de lista para almacenar dígitos en una `base` específica, utilizando las funciones `succ`, `pred` y `plus` para encapsular la lógica aritmética.
Demuestra la aplicación de recursión y condiciones (`cond`) para el tratamiento posicional de las cifras y la gestión de acarreos (carry) o préstamos (borrow).
Este ejemplo es clave para comprender la `Abstracción de Datos` y la creación de Tipos de Datos Abstractos (TDA) que ocultan su implementación subyacente.

######## CODIGO ########
#lang eopl
(define base 16)
(define zero '())
(define zero? null?)

(define succ
  (lambda (n)
    (cond
      [(zero? n) (list 1)]
      [(= (car n) (- base 1)) (cons 0 (succ (cdr n)))]
      [(cons (+ 1 (car n)) (cdr n))])))
    
(define pred
  (lambda (n)
    (cond
      [(zero? n) (eopl:error "El número no puede ser cero")]
      [(= (car n) 0) (cons (- base 1) (pred (cdr n)))]
      [(equal? (succ zero) n) '()]
      [else (cons (- (car n) 1) (cdr n))])))

;Area del programador
(define plus
  (lambda (x y)
    (if
     (zero? y)
     x
     (succ (plus x (pred y))))))
