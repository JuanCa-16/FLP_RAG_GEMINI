####
**Introducción a los Árboles de Sintaxis Abstracta**

En este momento tenemos limitaciones con respecto a la presentación; vimos dos representaciones:
- Basada en listas
- Basada en procedimientos

Sin embargo, estamos limitados por el lenguaje de programación `Racket`. Vamos a utilizar los `Árboles de Sintaxis Abstracta`.

¿Qué son?
Son representaciones desde la gramática, donde se relaciona la estructura del tipo de dato con respecto a su especificación.

####
**Especificación de la gramática lc-exp**

La gramática para las expresiones del cálculo lambda (`lc-exp`) se define de la siguiente manera:

- `<lc-exp> ::= <identifier>`
  `var-exp(id)`
- `<lc-exp> ::= "lambda" "(" <identifier>* ")" <lc-exp>`
  `lambda-exp(lid, exp)`
- `<lc-exp> ::= "(" <lc-exp> <lc-exp> ")"`
  `app-exp(rator, rand)`

Se utilizan los constructores `var-exp`, `lambda-exp` y `app-exp` para representar estas estructuras.

####
**Representación de una expresión de variable**

A continuación se muestra la representación de una expresión de variable simple:

```racket
(var-exp 'x)
```

```mermaid

```

####
**Representación de una expresión lambda**

A continuación se muestra la representación de una expresión `lambda` que recibe una lista de identificadores:

```racket
(lambda-exp
  (list 'x 'y 'z)
  (var-exp 'y))
```

```mermaid
graph TD
    A["lambda-exp"] -->|lid| B["x, y, z"]
    A -->|exp| C["var-exp"]

```

####
**Representación de una expresión de aplicación (app-exp)**

A continuación se muestra una expresión de aplicación compuesta:

```racket
(app-exp
 (app-exp
  (lambda-exp
   '(a b c)
   (lambda-exp
    '(x y z)
    (var-exp 'x)))
  (var-exp 'z))
 (lambda-exp
  '(a b c d e f)
  (var-exp 'e)))
```


####
**Ventajas de la representación AST**

Esta representación permite:
- Saber si el código fuente está bien escrito.
- Representar los datos de una forma independiente de los lenguajes de programación.

####
**Estructura compleja de aplicación en AST**

Representación de una estructura de aplicación con múltiples niveles de anidamiento:

```racket
(app-exp
 (app-exp
  (app-exp
   (app-exp
    (var-exp 'x)
    (lambda-exp
     '(a b c)
     (var-exp 'y)))
   (app-exp
    (lambda-exp
     '(x y z)
     (var-exp 'p))
    (lambda-exp
     '(m n o p q r t u)
     (lambda-exp
      '(v w x y z)
      (lambda-exp
       '(a b c)
       (var-exp 'p))))))
  (app-exp
   (var-exp 's)
   (var-exp 'r)))
 (app-exp
  (app-exp
   (var-exp 'a)
   (var-exp 'b))
  (var-exp 'x)))
```
