#### **Análisis de tipos en expresiones con procedimientos**

Considere la siguiente expresión:

```racket
let
  j = proc (int x, ? y) +(x, y)
  t = proc (? k, int a, ? b) (k a b)
in
  (t j 2 3)
```

Indique su tipo.

**Respuesta**

- `y` = `int`
- `k` = (`int` * `int` -> `int`)
- `b` = `int`

La expresión es de tipo `int`.

#### **Análisis de tipos en procedimientos de orden superior y expresiones anidadas**

Considere la siguiente expresión:

```racket
let
  j = proc (int x, ? y)
    if (y 2)
    then +(x, 2)
    else -(x, 3)
  t = proc (? k, int a, ? b, ? c) (k +(a, b) c)
  s = proc (? a) zero?(a)
in
  let
    p = proc (? m, ? n) (m n 5 14 s)
  in
    (p t j)
```

Indique su tipo.

**Respuesta**

- `y` = (`int` -> `bool`)
- `k` = (`int` * (`int` -> `bool`) -> `int`)
- `b` = `int`
- `c` = (`int` -> `bool`)
- `a` = `int`
- `m` = ((`int` -> `bool`) * `int` * `int` * (`int` -> `bool`) -> `int`)
- `n` = (`int` * (`int` -> `bool`) -> `int`)

La expresión es de tipo `int`.
